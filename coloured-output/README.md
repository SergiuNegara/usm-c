# Evidențierea rezultatului la consolă
## Sarcină
Se dau numerele naturale: n, A<sub>0</sub>, A<sub>1</sub>, ... A<sub>n</sub>. Evidențiați numerele:
1. divizibile cu 5
2. impare și negative
3. care satisfac condiția: | A<sub>i</sub> | < i<sup>2</sup>
